About Me

I am a PhD student in Deren Eaton's lab at Columbia University.

I like plants, birds, music, and side projects. This site is designed to be a central repository for blogs and creative projects, while also having information about my qualifications -- sort of a more personal, flexible LinkedIn alternative.

I want this site to be home to blogs, protocols, pictures, and miscellaneous projects: recipes, writing, and music.

About this site:

This website is served with Apache from a Raspberry Pi device in my home. I maintain the website with GitHub. The underlying code is based on an html template -- "Black & White" -- from ThemeWagon. My main intention for the site design was to be as lightweight as possible (after all, the Raspberry Pi can only do so much!), while still being something that I find appealing.
